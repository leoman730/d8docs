    Configure xdebug with phpstorm for docksal

    Follow instruction on Using Xdebug debugger

    Setting for the docksal.yml 

      cli:
    hostname: cli
    image: ${CLI_IMAGE}
    volumes:
      - project_root:/var/www:rw
      - docksal_ssh_agent:/.ssh-agent:ro
    environment:
      - XDEBUG_ENABLED=1
      - XDEBUG_CONFIG=idekey=PHPSTORM remote_host=192.168.10.1
      - PHP_IDE_CONFIG=serverName=${VIRTUAL_HOST}

    Module upgrade helper

    There’s useful module Drupal Module Upgrader can be used to identify compatibilitiess issue D7 modules.

    # To generate an issue report
drush dmu-analyze  --skip=info --path=modules/custom/nyulaw_authentication nyulaw_authentication

# To try automatically fix issue
drush dmu-upgrade  --skip=info --path=modules/custom/nyulaw_authentication nyulaw_authentication
