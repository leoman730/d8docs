    Testing

    Testing list from Clayton

    • Drupal Migration Compoenents

    Access control

    • Make sure a user who doesn’t has access right can not edit the piece of the content
    • If a user has ‘full access control’ priviledge, she should be able to access all content for editing

    Protected Page (module)

    • Make sure page that is passwd protected still working

    Calendar Block

    • Calendar Homepage block should only display ‘Ping to homepage event’
    • Cache durarion setting in the admin should reflect in the calendar block.
      • Make sure drupal page load not hitting Coldfusion everytime

    Coldfusion Authentication

    • Make sure only login user can view the content
    • Make sure once user logout, content viewing need to be re-authenticated
    • ANONYMOUS user can not see unpublished content
    • Unpublished menu should not show up on menu tree

    Faculty Services

    • Faculty profile could pull data from Faculty service (faculty view, faculty flexslider view)
      • http://d8.law.nyu.edu/faculty_flexslider_service_view?facultyid=20315&items_per_page=1&_format=json
      • http://d8.law.nyu.edu/faculty_service_view?facultyid=20315&items_per_page=1&_format=json

    News/Press highlight

    • There should be a redirect icon for news/press highlight that link to external page.

    Basic Page

    • Sidebar blocks
      • When the node has sidebar block content, it should show on the right sidebar. (http://d8.law.nyu.edu/academics)
      • When the node doesn’t have sidebar block content, it should not show on the right sidebar. (http://d8.law.nyu.edu/academics/lawyeringprogram), and the page content should have full width on the right.

    Metatag

    * Make sure metatag is set correctly (meta description, title etc...)

    Paragraphs

    * Make sure basic editor can not create new Paragraphs by type defined in the permission list

    When a paragraphs row is disabled I should see overlay text ‘Disabled’ on the editorial page

    When a paragraphs row is disabled I should not see the row content in the frontend view

    Leftside menu

    * Menu item that children [Areas of Study](https://d8.law.nyu.edu/areasofstudy)
* Menu item that without child [Faculty](https://d8.law.nyu.edu/areasofstudy/human-rights/faculty)
* Page doesn't have menu accsociate with [Checklist](https://d8.law.nyu.edu/graduateaffairs/incomingstudents/checklist)

    Centers

    * Make sure Centers header template is render correctly
    * /centers/civiljustice
    * /centers

    Layout option

    When a page flag transparent header I should see transparent header in the view

    When a page has a hero image I should see the hero image in the dynamic lead region

    When I on ‘Advanced Page’ Editorial page I should have layout option

    When I on ‘News Item’ Page Editorial page I should have layout option

    When a page has a page class defined I should see the page class within the html tag

    When I login as an Advanced User or Basic Editor I should not see layout option on Advanced page content type

    When I login as an Fullsite editor or Admin I should see layout option on Avdanced page content type

    Flexslider

    When I am on a mobile device Flexslider should be paused by default

    When I am on a Desktop computer Flexslider should be played by default

    Display title

    When display_title is not empty It should replace the value of the title field on node page view

    When display_title is set to [‘notitle’] It should not display title on the node page view

    When display_title is empty It should see default title value in the node view
