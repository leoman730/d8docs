    Elastic Search

    Configuration for docksal

    The image from dockerhub for ES has deprecated, and Elastic search image now can be pull to docksal directly from elastic.co 

      elasticsearch:
    hostname: elasticsearch
    image: docker.elastic.co/elasticsearch/elasticsearch:6.4.0
    ports:
      - "9200:9200"
    dns:
      - ${DOCKSAL_DNS1}
      - ${DOCKSAL_DNS2}

    There’s a catcha for setting vm.max_map_count. Refer to: https://www.elastic.co/guide/en/elasticsearch/reference/6.4/docker.html

    docker-machine ssh docksal
sudo sysctl -w vm.max_map_count=262144

    Setting ports 

    fin vm stop
VBoxManage modifyvm "docksal" --natpf1 "tcp-port5602,tcp,,5602,,5602";
VBoxManage modifyvm "docksal" --natpf1 "tcp-port9200,tcp,,9200,,9200";
fin vm start

    Elasticsearch web access: x-Pack security: elastic/changeme

    http://192.168.64.100:9200/ http://192.168.64.100:5601/
